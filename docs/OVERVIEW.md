# Overview — کلینیک قلب دکتر بهنام یغمایی

## Product (high level)

A full clinic management system for a cardiology practice:

| Layer | Scope |
|:---|:---|
| Public website | Home, about, services, blog, contact, online appointment, waiting display |
| Patient portal | Login + patient dashboard |
| Admin panel | Patients, appointments, queue, finance, insurance, SMS, reminders, settings, backup |
| API | REST + JWT for mobile/integrations |

## Stack (public facts only)

- PHP 8.3 native OOP / MVC
- MySQL
- Bootstrap 5 RTL + Vazirmatn
- Jalali calendar throughout
- PWA, SEO, Dark Mode
- Security: CSRF, rate limit, 2FA, roles/permissions, security headers

## What this showcase includes

- Landing page (GitHub Pages)
- Static HTML/CSS UI demo matching the real design system
- Sample admin dashboard / patients / queue screens

## What is intentionally excluded

- PHP application source
- Migrations, seeds, `.env`
- Real booking, payment, SMS, and JWT logic
- Production credentials
